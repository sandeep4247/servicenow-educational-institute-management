## Overview
This project is a custom ServiceNow application designed to simulate an educational institute management system.  
It automates student admission processes, fee calculation, and student academic progress tracking.

The application demonstrates practical implementation of ServiceNow configurations such as custom tables, client scripts, UI policies, and role-based access control.

---

## Features

- Student Admission Management
- Automated Fee Calculation Based on Grade
- Student Progress Tracking
- Automated Calculation of Total, Average, and Percentage
- Role-Based Access Control
- UI Policies to control field behavior
